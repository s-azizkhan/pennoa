data philosophy

local. private. open.
always.

0

network calls

Pennoa runs identically with the network unplugged. There is no API to phone home to.

0

accounts

No sign-up, no email, no ID. Open the app, add a meeting, see a banner. That is the whole flow.

0

telemetry

Nothing leaves your machine. Not a crash report, not a feature ping, not a usage metric.

1

file

Your meetings live in a single human-readable JSON file in your app-data directory. Back it up. Diff it. Own it.

roadmap

where pennoa is going.

  1. phase 1

    core experience

    Manual add. Banner overlay. T−10 and T−5 triggers. Menu bar icon. Auto-start at login. Eleven banner styles. Eleven aircraft. Reduce-motion fallback.

  2. phase 2

    import & control

    .ics drag-and-drop. Pause for an hour or until tomorrow. Daily / weekly repeats. Custom reminder intervals.

  3. phase 3

    polish

    Subtle sound cues. Next-up tray badges. Focus / Do Not Disturb awareness. Sparkle and WinSparkle auto-update.

  4. phase 4

    sync & advanced recurrence

    CalDAV without a third-party middleman. Complex recurrence (every third Tuesday). Multi-device sync via a local shared folder.
